## Installation

```bash
$ npm install
```

## Running the app

```bash
# Run database
$ sudo docker-compose up

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod

To view available endpoints go to http://localhost:5000/api

```
